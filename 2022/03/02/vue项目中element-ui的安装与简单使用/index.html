element-ui的安装

安装好之后在package.json的<dependencies>标签下会增加element-ui依赖。

element-ui的简单使用

  1. 在main.js中增加导入和让Vue使用ElementUI
  2. 在Element UI 官网中有丰富的代码示例，复制一段某组件的代码粘贴到组件中的相应位置，运行后在浏览器进入到相应页面查看页面展示效果。
