Viewing the Commit History (git log)

[Abstract]
We can use git log command to view the commit history in a Git repository.
We can also add some additional options to specify the log format (such as --pretty=oneline and --pretty-format) or filter the commit entry (such as -<n>).

git log

git log: Lists the commit history in reverse chronological order(时间倒序).

For each commit:

  1. Commit hash
  2. Author name and Author email
  3. Author date
  4. Commit message

Options

-p/-patch

Show the difference introduced in each commit in addation to the information shown in git log.

For each commit:

  1. Commit hash
  2. Author name and Author email
  3. Author date
  4. Commit message
  5. Differences introduced in this commit

Example:

-stat

Show some abbreviated stats(统计数据) for each commit in addation to the message shown in git log.

For each commit:
(insertion and deletion is talking about lines added and removes)

  1. A list of modified files, each list entry (“Rakefile | 2 +-“ for example) shows:
  • a changed file name (Rakefile)
  • how many insertions and deletions in total (2)
  • how many insertions, one insertion displayed a ‘+’
  • how many deletions, one deletion displayed a ‘-‘
  1. A summary information:
  • how many files are changed
  • how many insertions in total
  • how many deletions in total

Prebuilt Format

There are four prebuilt values for --pretty option(“oneline”, “short”, “full” and “fuller”).

--pretty=oneline

  1. Commit hash

Limit Options

limit options: options that let you show only a subset of commits

Number Limit: -<n>

-<n>: Show only the last n commits.

Time Limit

--since=<date>/--after=<date>: Limit the commits to those made after the specified date.
--until=<date>/--before=<date>: Limit the commits to those made before the specified date.

Option Value: a specified date

--since=2.weeks: List commits made in the last 2 weeks.
--since=2.years: List commits made in the last 2 years.
--since="2008-01-15": List commits made after 2008-01-15.
--since="2 years 1 day 3 minutes ago": List commits made in the last 2 years, 1 day and 3 minutes.

Filter

--author=<pattern>, --committer=<pattern>

List commits whose author/commiter matches the specified pattern.

With more than one --author=<pattern>, list commits whose author/committer matches any of the given patterns.

--grep=<pattern>

List commits whose commit message matches the specified pattern.

With more than one --grep=<pattern>, list commits whose commit message matches any of the given patterns.

Add a --all-match option after a series of --grep=<pattern> will list commits whose commit message matches all of the given patterns.

-S <string>

List commits that has added or removed this string.

For example: -S function_name shows commits that has removed a reference of a specific function.

-- <path>

List commits that has changed a specific file or directory.

This option is always the last option of a command.

--no-merges

List commits that are not merge commits.
