Remote Repository

  1. Connect local repository with a remote repository:
    $ git remote add origin git@github.com:ZhuYuxi333/learngit.git
    “origin” is the default name Git gives to a remote repository.
  2. View the connected remote repositories of a local repository: git remote -v
  3. Push the contents in local repository to its related remote repository:
    • The first time: git push -u origin main
    • Later: git push origin master
  4. Disconnect local repository with a remote repository:
    $ git remote rm origin
