Undoing Things

Redo Commit: git commit --amend

How to redo the last commit?

  1. Make changes in the working directory.
  2. Stage the changes.
  3. Commit again using git commit --amend.

git commit --amend will:

  1. Take the changes in the staging area and add it to the last commit.
  2. Fire up the commit-message editor of the last commit for you to overwrite with a improved commit message.

Unstage a Staged File

  • git reset HEAD <file>
  • git restore --staged <file> (Git version above 2.23.0)

Unmodify a Modified File:

Revert back to what it looked like in the last commit.

  • git checkout -- <file>
  • git restore <file> (Git version above 2.23.0)
