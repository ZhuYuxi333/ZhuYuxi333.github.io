2. 各部分详解

2.1 标题

HTML标题（heading）是通过<h1> ~ <h6>标签来定义的。

2.2 段落

HTML段落（paragraph）是通过标签<p>来定义的。

2.3 链接

HTML链接是通过标签<a>来定义的。

2.4 图像

HTML图像（image）是通过标签 <img> 来定义的。注意： 图像的名称和尺寸是以属性的形式提供的。

2.5 表格

HTML表格通过<table>标签来定义：

  • 每一个表格包含若干行，每一行由一对<tr>标签(table row)定义
  • 表格中的每一行又分为若干单元格，其中：
    • 每一个表头单元格由一对<th>标签(table head)定义
    • 每一个数据单元格由一对<td>标签(table data)定义，数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。
  • 表格边框：如果不定义边框属性，表格将不显示边框。如果希望显示边框，需要定义表格的边框属性的属性值为1：<table border="1"> ... </table>。

2.6 区块

块级元素和内联元素：

  • 块级元素：在浏览器中显示时，通常会以新行来开始（和结束）。比如：<h1>, <p>, <ul>, <table>。
  • 内联元素：在浏览器中显示时，通常不会以新行开始。比如：<b>, <td>, <a>, img。

<div>标签

<div>标签用于组合块级元素，以便通过 CSS 来对这些元素设置样式。


在网页设计中经常使用<div>标签和CSS来对网页进行布局：

<span>标签

<span>标签用于组合行内元素，被 元素包含的文本，可以使用 CSS 对它定义样式，或者使用 JavaScript 对它进行操作。

2.7 表单

HTML表单用于收集用户的输入，表单使用<form>标签来定义。

表单中可以包含若干表单元素，表单元素允许用户在表单中输入内容。输入元素使用标签<input>来定义，类型属性(type)决定输入类型。

文本域、密码字段、单选按钮、复选框

提交按钮（Submit Button）

<input type="submit"> 定义了提交按钮。当用户单击确认按钮时，表单的内容会被传送到另一个文件。表单的动作属性（action）定义了目的文件的文件名。由动作属性定义的这个文件通常会对接收到的输入数据进行相关的处理。

下面的整个结构称为“规则集”

各部分释义如下：

  • 选择器（Selector）：HTML 元素的名称位于规则集开始。它选择了一个或多个需要添加样式的元素（在这个例子中就是 p 元素），如果选择器中有多个元素则不同元素之间用逗号,分隔。要给不同元素添加样式只需要更改选择器就行了。
  • 声明（Declaration）：对选择器的一条样式规则，由一对属性+属性值组成，每一条声明中属性与属性值用冒号:分隔，不同的声明用分号;分隔。

3. 选择器

选择器有许多不同的类型。上面只介绍了元素选择器，用来选择 HTML 文档中给定的元素。但是选择操作可以更加具体。下面是一些常用的选择器类型：（更多信息请参考：选择器）

选择器名称 选择的内容 示例
元素选择器（也称作标签或类型选择器） 所有指定(该)类型的 HTML 元素 p
选择 <p>
ID 选择器 具有特定 ID 的元素（单一 HTML 页面中，每个 ID 只对应一个元素，一个元素只对应一个 ID） #my-id
选择 <p id="my-id"> 或 <a id="my-id">
类选择器 具有特定类的元素（单一页面中，一个类可以有多个实例） .my-class
选择 <p class="my-class"> 和 <a class="my-class">
属性选择器 拥有特定属性的元素 img[src]
选择 <img src="myimage.png"> 而不是 <img>
伪（Pseudo）类选择器 特定状态下的特定元素（比如鼠标指针悬停） a:hover
仅在鼠标指针悬停在链接上时选择 <a>。

5. 盒模型

页面里大部分 HTML 元素都可以被看作若干层叠的盒子。CSS 布局主要就是基于盒模型的。每个占据页面空间的块都有这样的属性：

  • padding：即内边距，围绕着内容（比如段落）的空间。
  • border：即边框，紧接着内边距的线。
  • margin：即外边距，围绕元素外部的空间。

  • width ：元素的宽度
  • background-color ：元素内容和内边距底下的颜色
  • color ：元素内容（通常是文本）的颜色
  • text-shadow ：为元素内的文本设置阴影
  • display ：设置元素的显示模式（暂略）
