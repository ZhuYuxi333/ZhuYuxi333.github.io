在Mac系统中安装Vue并使用vue-cli创建第一个vue项目

本文记录了在Mac系统上配置Vue环境，并成功创建运行第一个Hello World项目的过程。

  • 安装vue
  • 创建vue项目

参考教程

安装vue

  1. 查看node.js和npm版本
  2. 安装淘宝镜像
  3. 安装webpack
  4. 安装vue脚手架
  5. 查看是否安装成功

创建vue项目

  1. 创建文件夹VueProjects，之后所有的vue项目都放在此目录下；进入该目录，创建文件夹VueLearningProjects用于存放所有的学习示例程序。
  2. 进入目录:cd ~/VueProjects/VueLearningProjects，在该目录下执行下面的语句初始化一个vue项目，项目名为vue_project：
  3. 初始化项目完成后在 “~/VueProjects/VueLearningProjects” 目录下出现了一个名为vue_project的目录，该目录即为vue_project项目的根目录，进入该目录：cd vue_project
  4. 安装项目依赖：$ sudo npm install
  5. 启动项目：$ npm run dev (关闭：control+C)
  6. 访问 http://localhost:8080 打开vue项目
